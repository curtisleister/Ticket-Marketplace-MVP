# Stub Vault — MVP

**Your Seat. Your Story. Your Stub.**

A licensed sports ticket memorabilia marketplace MVP. Turn your digital ticket into a premium physical souvenir — officially licensed, personally yours.

## Quick Start

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS v4** for styling
- No backend — all mock data for demo purposes

## Screens

1. **Landing Page** — Hero, how-it-works, featured partnerships, social proof
2. **Browse Teams** — Filterable grid of partner teams by league
3. **Select a Game** — Available games for the selected team
4. **Upload Ticket** — Drag-and-drop upload with simulated verification
5. **Ticket Preview** — Premium physical ticket preview with pricing
6. **Checkout & Confirmation** — Shipping form and order confirmation

## Build for Production

```bash
npm run build
```

Output goes to `dist/`.
